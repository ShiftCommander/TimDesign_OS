# System AI Entry Rules

## Principle

AI-generated content is disposable by default.

An AI idea, draft, template, or rule may enter the OS only if it has a clear role.

## Entry Criteria

Before adding AI-generated content, confirm:

- Destination: where exactly does it belong?
- Use: what concrete task will it support?
- Action value: does it help Tim act, sell, deliver, decide, or publish?
- Replacement: does it replace or improve an existing file?
- Shelf life: will it still be useful in 30 days?
- Owner: does it belong in OS, Notion, a client folder, or nowhere?

## If Temporary

Put it in `00_Inbox` with:

- why it exists;
- what should happen next;
- expiry/review date.

## Rejection Rule

Do not add content that is vague, duplicative, speculative, motivational, over-structured, or disconnected from active TimDesign execution.
