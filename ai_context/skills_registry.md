# Capability Registry

Purpose: list reusable work methods that are already useful in TimDesign work.

## Use rule

Check this file before inventing a new method for positioning, site structure, copy, design critique, audits, or agent handoff.

After a good result, add a short note with the method used and when to reuse it.

## Current methods

### Product Marketing
Status: active
Use for audience, positioning, objections, proof, and message strategy.

### Site Architecture
Status: active

### Copywriting
Status: active

### Ogilvy
Status: active

### TimDesign Copy Brain
